Crowdox project information directly on your site

== Description ==

"BC Crowdox widget" allows you to pull single <a target="_blank" href="https://crowdox.com/">Crowd OX</a> project on your site using widget or shortcode.
Comes with simple style but you can restyle it as you desire.

Promote your Crowdox funding project on your Wordpress website and help drive traffic to your Crowdox campaign.
- Show project image and short description
- Displays the number of backers, current amount funded, percentage funded and days left on project.
- Customizable widget title name


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/crowdox-project-tracker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearance/Widgets, find widget "BC Crowdox widget" and add to desired sidebar 
4. You can use shortcode instead of a widget: [crowdox url= &quot; your project url &quot; ]
